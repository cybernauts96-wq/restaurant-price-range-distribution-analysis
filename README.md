# 💰 Price Range Distribution Analysis

## 📌 Overview
This project analyzes the distribution of restaurant price ranges using a cleaned restaurant dataset. It calculates the number and percentage of restaurants in each price range, exports the results to a CSV file, and creates visualizations for better understanding.

This task was completed as part of my **Data Analyst Internship – Level 1 (Task 3)**.

---

## 🎯 Objectives
- Analyze the distribution of restaurant price ranges.
- Calculate the percentage of restaurants in each price category.
- Identify the most and least common price ranges.
- Export the analysis results to a CSV file.
- Create professional visualizations.

---

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib

---

## 📂 Dataset
- **Input:** `cleaned_restaurants.csv`
- The dataset contains restaurant information such as city, cuisines, ratings, and price range.

---

## 📊 Analysis Performed
- Counted restaurants in each price range.
- Calculated the percentage distribution.
- Identified:
  - Most common price range
  - Least common price range
- Exported the results to a CSV file.
- Created:
  - Bar Chart (Restaurant Count by Price Range)
  - Pie Chart (Percentage Distribution)

---

## 📁 Output Files
- `price_range_distribution.csv` – Summary of price range counts and percentages.
- `price_range_distribution.png` – Visualization of the analysis.

---

## ▶️ How to Run
1. Clone the repository.
2. Install the required libraries:

```bash
pip install pandas numpy matplotlib
```

3. Place `cleaned_restaurants.csv` in the project folder.
4. Run the script:

```bash
python price_range_distribution.py
```
